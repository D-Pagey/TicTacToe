TicTacToe

This is an Advanced Front End Developer project from the FreeCodeCamp curriculum. The user stories to be completed are:

  • I can play a game of Tic Tac Toe with the computer.
  • My game will reset as soon as it's over so I can play again.
  • I can choose whether I want to play as X or O.

This was made using: Flexbox and native JS. No CSS or JavaScript frameworks or libraries were used. The code can be seen on Github.
